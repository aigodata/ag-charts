快速上手

前言

ag-charts是站在巨人的肩膀上, 底层使用了大家熟悉图表库(例:echarts v3.8), 我们优化了展示. 对图表进行了封装.

自我介绍

ag-charts尽量避免重复发明轮子, 浪费研发资源, 支持模块化加载. 方便换肤.
